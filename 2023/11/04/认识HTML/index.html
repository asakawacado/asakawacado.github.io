1.HTML文档基本格式

​ 下面是一段新建HTML文件的默认源代码

​ <DOCTYPE> :文档类型声明 <html>:根标记 <head>:头标记 <body>:主体标记

  1. <DOCTYPE>: 位于文档最前面，用于向浏览器说明当前文档用哪种HTML或XHTML标准规范。
  2. <html>: 位于 后面，也称根标记，用于告知浏览器自身是一歌HTML文档<html>标记着HTML文档的开始</html>标记着GTML文档的结束，在它们之间的是文档的头部和主题内容。
  3. <head>: 用于定义HTML文档的头部信息，也称头部标记，紧跟在标记之后，主要用来封装其他位于文档头部的标记
  4. <body>:用于定义HTML文档所要显示的内容，也称主体标记。浏览器显示的图像、音频、和视频等信息都必须位于<body>标记内。

2.HTML标记

​ 在HTML页面中，带有< >符号的元素被称为HTML标记或HTML元素，如上述<html> <head> <body>都是HTML标记。HTML标记主要有单标记和双标记。

  1. 单标记:
    <标记名/>: 例<hr/>，用于定义一条水平线 。

  2. 双标记

    ​ <标记名> 内容 </标记名>: 例<h3>三级标题</h3> 用于表示一个三级标题。

  3. 注释标记

    ​ <!--注释语句-->: 特殊的标记，用于注释。

3.标记的属性

​ 在使用HTML制作网页时，仅仅使用标记是很单调的，因此我们引入一个属性的概念，通过对标记属性的修改，使网页元素多样化。如下例子:

​ <h1 align="center">文本</h1>: 通过修改h1标签的align属性，达到使文本居中的效果，这种修改通常是以键值对的形式设定的key = "value",即属性的关键字key和属性的数值value。

​ 同时，为了能够灵活的，粒度小的修改各个元素的属性，允许使用嵌套去对已修改标签内部进行覆盖式修改。

4.HTML 文档头部相关标记

​ HTML文档头部相关标记是指head标记内的一些标记，如<title> <meta/>等。

  1. 设置页面标题标记<title>

    ​ <title>标记用于设置网页标题名称 ， 如<title>网页名称</title>

  2. 定义页面元信息标记<meta/>

    ​ <meta/>通过使用成对的**[名称|值]**使用其属性定义页面相关参数，例如为搜索引擎提供网页的关键字、作者姓名、内容描述，以及定义网页的刷新时间等。下面介绍<meta/>标记常用的几组设置。

    1. <meta name="名称" content="值"/> 在<meta/>标记中**[name|content]属性可以为搜索引擎提供信息，其中name提供搜索内容名称，content**提供对应搜索内容值。

      • 设置网页关键字

        <meta content="搜索内容值1,搜索内容值2,.....搜索内容值n" name="keywords">其中搜索内容值可以用,分隔。

      • 设置网页描述

        <meta content="网页描述...." name="description">

      • 设置网页作者

        <meta content="作者名称" name="author">

    2. <meta http-equiv="名称" content="值"/ 在<meta/>标记中，使用**[http-equiv|content]属性可以设置服务器发送给浏览器的HTTP头部信息，为浏览器显示该页面提供的相关参数。其中http-equiv属性提供参数类型，content**提供对应参数值。默认会发送<meta http-equiv="Content-Type" content="text/html"/>通知浏览器发送的文件类型是HTML。具体应用如下。

      • 设置字符集

        <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />

        其中content的值为text/html和charset =utf-8中间用;隔开，用于说明当前文档类型为HTML字符集为utf-8。

      • 设置页面自动刷新与跳转

        <meta http-equiv="refresh" content="10;url=http://www.xxxx.xxx" />其中http-equiv属性的值为fresh，content属性的值为数值和url地址，中间用;隔开，用于指定在特定的时间后跳转至目标页面，时间单位默认为秒。

  3. 引用外部文件标记<link>

    ​ 在<head>中使用<link>可引用外部文件，一个页面允许使用多个<link>标记引用多个外部文件。

  4. 内嵌样式标记<style>

    ​ <style>标记用于为HTML文档定义样式信息，位于<head>头部标记中。在HTML中使用style标记时，常常定义其属性为type，相应的属性值为text/css，表示使用内嵌式的CSS样式。
